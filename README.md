## Problem Covered

### 1. Contiguous Array (LeetCode 525)
**Difficulty:** Medium  
**Topic:** HashMap, Prefix Sum

**Approach:**
- Convert `0` to `-1`
- Use prefix sum to track balance
- Store first occurrence of each sum using HashMap

**Complexity:**
- Time: O(n)
- Space: O(n)
